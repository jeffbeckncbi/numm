Comparing BITS 1.0 Test against JATS 1.1d3

Summary

BITS 1.0 Test IS NOT compatible with JATS 1.1d3. Details are below.

Attribute Type Changes

  • @continued-from attribute type was "IDREF" but is now "CDATA".

Superstructure Changes

  • <body> was a SUPERSTRUCTURE element in JATS 1.1d3 model but is not in BITS 1.0 Test.
  • <conf-date> has become a SUPERSTRUCTURE element in BITS 1.0 Test but it was not SUPERSTRUCTURE in JATS 1.1d3.

Element Only or #PCDATA/EMPTY Changes

  • <pub-date> had an ELEMENT ONLY model in JATS 1.1d3, but it has a #PCDATA/EMPTY model in BITS 1.0 Test.
  • <prefix> had a MIXED or EMPTY model in JATS 1.1d3, but it has an ELEMENT model in BITS 1.0 Test.

Section Model Changes

  • <sec> had a section model in JATS 1.1d3, but it does not in BITS 1.0 Test.
  • <rb> did not have a section model in JATS 1.1d3, but it has a section model in BITS 1.0 Test.
